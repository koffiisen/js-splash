JS SPLASH

Very easy to use js splash screen. You don't need jQuery, Angular, React, Bootstrap, etc. Also included is a minimized version of the js. For the highest performance, bolt the minimized JS directly into your HTML app so that you avoid yet another HTTP request. 

//Get started
                    <script src="dist/js-splash-2.0.min.js"> </script>
